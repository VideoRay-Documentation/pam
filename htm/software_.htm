Software Guide

This section contains information on how to configure your software setup for your serial device, and how to use Cockpit to access information about the PAM board and attached devices.

Virtual Communication Ports

The PAM makes use of Cockpits built in virtual port functions. You can learn more about virtual ports from the Pro 4 manual. If you wish to modify the default ports or add new virtual ports, it is highly recommended that you read this section.
