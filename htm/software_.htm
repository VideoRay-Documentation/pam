Software Guide

This section contains information on how to configure the PAM for your serial device, and how to use VideoRay Cockpit to access information about the PAM board and attached devices.

Virtual Communication Ports

For pass through communications, the PAM makes use of virtual ports. You can learn more about virtual ports from the Pro 4 manual. If you wish to modify the default ports or add new virtual ports, it is highly recommended that you read this section.
