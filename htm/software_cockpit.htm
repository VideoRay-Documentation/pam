VideoRay Cockpit Configuration

VideoRay Cockpit requires an accessory device configuration file to be installed to use a PAM enabled device.

The accessory device configuration file informs VideoRay Cockpit how to address the PAM (node id) as well as other parameters. If the accessory device configuration file is not installed VideoRay Cockpit will not communicate with the PAM.

Accessory device files can be created using the vrDeviceConfiguration tool. However, a factory configured file is typically provided with the PAM enabled accessory purchased from VideoRay.

PAM accessory Device configuration files provided must be installed into the C:\Users\VideoRay\AppData\Local\VideoRay\AccessoryDevices\ folder. The accessory device configuration file is an XML file with human readable file names.

There are two fields which may require user modification, in which case a plain text editor such as notepad can be used. The file is formatted as such:

<PortName>
	<SimpleName>COM2</SimpleName>
	<FriendlyName>COM2</FriendlyName>
</PortName>
<BaudRate>600</BaudRate>

The PortName field designates which serial port will be used to relay the data. This can be a real or virtual comm port. The BaudRate field sets the baud rate of the Relay port.

Cockpit Diagnostics

VideoRay Cockpit software allows you to monitor the status of all connected PAM devices in real time.

  • First click the Maintenance and Diagnostic button on the launch bar in Cockpit.

  • Next, press the show communications diagnostic window button on the service bay screen, and the screen above will appear, allowing you to view data on your attached PAM devices.
